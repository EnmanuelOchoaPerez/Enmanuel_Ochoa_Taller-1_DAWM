Fundamentos de CSS

Hojas de Estilo en Cascada: La capa de presentación de la Web.

1. ¿Qué es CSS y para qué sirve?

CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico utilizado para describir la presentación de un documento escrito en un lenguaje de marcado (como HTML).

¿Para qué sirve? Su propósito principal es separar el contenido de la estructura (HTML) del diseño visual (CSS). Permite controlar colores, fuentes, espaciados, tamaños, disposición (layout) y la adaptación del diseño a diferentes tamaños de pantalla (diseño responsivo).

2. Anatomía de una Regla CSS

El código CSS se compone de "reglas". Una regla CSS dicta cómo se debe ver un elemento específico.

selector {
    propiedad: valor;
}
  • Selector: Es el patrón que le indica al navegador a qué elemento(s) HTML se le aplicará el estilo.
  • Propiedad: Es la característica específica del elemento que queremos modificar (ej. color, font-size, margin).
  • Valor: Es el estado o medida que le asignamos a esa propiedad (ej. red, 16px, auto).

3. Formas de insertar CSS

Existen tres métodos principales para aplicar CSS a un documento HTML:

A. CSS en línea (Inline)

Se aplica directamente en la etiqueta HTML usando el atributo style. No se recomienda para uso general porque mezcla contenido con diseño.

<h1 style="color: blue;">Título Azul</h1>

B. CSS Interno

Se coloca dentro de la etiqueta <style> en la sección <head> del documento HTML. Útil para páginas únicas de estilo particular.

<head>
  <style>
    p { color: green; }
  </style>
</head>

C. CSS Externo

Es la mejor práctica. Se escribe el código en un archivo independiente con extensión .css y se vincula al HTML mediante la etiqueta <link> en el <head>.

<link rel="stylesheet" href="style.css">

4. Tipos de Selectores Principales

Los selectores permiten apuntar a los elementos con gran precisión.

Selector Universal (*)

Aplica a todos los elementos de la página.

* { margin: 0; }

Selector de Etiqueta

Apunta a todos los elementos de un tipo específico.

p { font-size: 16px; }

Selector de Clase (.)

Apunta a elementos con un atributo class específico. Es reutilizable.

.destacado { font-weight: bold; }

Selector de ID (#)

Apunta a un elemento con un id único. No debe repetirse en la página.

#cabecera { background: gray; }

Selector Agrupado (,)

Aplica la misma regla a varios selectores distintos separados por coma.

h1, h2, h3 { color: navy; }

Selector Descendente ( )

Apunta a elementos que están dentro de otro (sin importar la profundidad).

div p { color: red; } /* P dentro de un DIV */

Selector Hijo Directo (>)

Apunta solo a los elementos que son hijos inmediatos de otro.

ul > li { list-style: none; }

Selector de Atributo ([])

Apunta a elementos que posean un atributo específico.

input[type="text"] { border: 1px solid black; }
