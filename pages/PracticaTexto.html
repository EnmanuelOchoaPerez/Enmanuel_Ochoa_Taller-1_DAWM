Bloques de Texto Básicos

La etiqueta <p> se utiliza para definir párrafos de texto independientes.
Si necesitamos realizar un salto de línea dentro del mismo párrafo sin crear uno nuevo, utilizamos la etiqueta <br> como acabamos de hacer aquí.

Formato y Énfasis del Texto

HTML ofrece múltiples etiquetas para modificar el significado y aspecto visual de las palabras:

Podemos destacar texto con fuerte importancia semántica (strong) o simplemente aplicarle un estilo en negrita visual (b).

Del mismo modo, podemos dar un énfasis especial a una palabra (em) o mostrarla en letra cursiva convencional (i).

Otras opciones de formato incluyen subrayar un texto (u) para llamar la atención, o utilizar la etiqueta para resaltar texto con un fondo brillante (mark), ideal para simulaciones de marcadores fluorescentes.

Para la "letra pequeña" de los contratos o notas al pie de página, se usa el elemento texto de tamaño reducido (small).

Si trabajas con matemáticas o química, puedes escribir variables elevadas usando un superíndice (sup) (ej: x2) o compuestos usando un subíndice (sub) (ej: H2O).

Para mostrar correcciones en un documento editorial, se puede marcar el texto eliminado (del) y el subsiguiente texto insertado en su lugar (ins).

Finalmente, si queremos agrupar unas palabras para aplicarles un comportamiento específico en línea sin alterar el bloque, usamos la etiqueta un contenedor neutro en línea (span).

Citas, Referencias y Abreviaturas

Cuando usamos siglas o acrónimos, la etiqueta HTML (abbr) permite añadir una descripción flotante que aparece al pasar el cursor.

Al hacer referencia al título de una obra, libro o película, se utiliza la etiqueta Don Quijote de la Mancha (cite).

Para incorporar citas textuales cortas dentro de un mismo párrafo, usamos la etiqueta esta es una cita en línea (q), la cual añade comillas automáticamente según el idioma del navegador.

Para citas textuales extensas que requieran su propio bloque independiente, se utiliza la estructura de bloque:

"Este es un ejemplo de bloque de cita (blockquote). Los navegadores web suelen aplicarle un margen izquierdo por defecto para separarlo visualmente del resto del contenido."

Texto Técnico y de Computación

Para mostrar fragmentos de código de programación dentro de una oración, usamos el elemento code (ej: String nombre = "Juan";).

Si queremos representar variables matemáticas o de desarrollo de software, recurrimos a la etiqueta x (var).

Para indicarle al usuario qué teclas debe presionar en su ordenador, usamos la etiqueta de teclado (kbd), por ejemplo: Presiona Ctrl + S para guardar.

Para simular o mostrar la salida de un programa informático o sistema, empleamos la etiqueta de muestra (samp): El servidor devolvió un Error 500: Internal Server Error.

Si lo que necesitamos es mostrar un bloque de código completo conservando de manera exacta todos los espacios en blanco y saltos de línea originales, la etiqueta de texto preformateado (pre) es la indicada:

// Bloque de texto preformateado (pre)

    console.log("¡Hola, mundo!");
}
